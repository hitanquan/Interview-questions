#### 1.html5 新特性？

- 语义化标签
  - 头部标签：<header>
  - 导航标签：<nav>
  - 内容标签：<article>
  - 定义文档某个区域：<section>
  - 侧边栏标签：<aside>
  - 尾部标签：<footer>
- 多媒体标签
  - 音频：<audio>
  - 视频：<video>

##### 什么是 dom？

- dom，即文档对象模型，可以看作是一套 w3c 组织提供的 web api，也可以理解为 dom 节点，就是 html 页面中的各种标签，这些 api 就是提供了许多函数来供开发人员操作网页上的 dom 节点（或者叫元素）。
