##### 1. 浏览器从输入 url 到展现页面发生了什么？

大致步骤：

1. 域名解析成 ip 地址。

2. 建立 tcp 连接（三次握手 🤝、四次挥手 🙋）。

3. 浏览器发起 http 请求，携带请求报文。
4. 服务器接收并处理请求，返回响应报文。
5. 浏览器渲染页面。

细节：

1. 域名是通过 【DNS 服务（也叫协议）】 来解析的，域名是为了方便给我们人类记忆，而计算机更擅长处理数字。
2. 什么是 tcp 连接，为什么要建立 tcp 连接：建立连接是为了确定需要通信的计算机（或其他网络设备）之间是否可以通信，确保通信的可靠性。
